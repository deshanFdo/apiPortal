# ============================================================================
# WSO2 API Manager Gateway Configuration
# ============================================================================
# This folder contains everything needed to run WSO2 API Manager as the
# API Gateway in front of the Express backend services.
#
# WSO2 API Manager provides:
#   - OAuth2 / JWT token validation
#   - Rate limiting (throttling tiers)
#   - API analytics and monitoring
#   - Developer portal for API discovery
#
# Usage:
#   cd wso2-gateway
#   docker-compose up -d
#
# Then publish the APIs via the WSO2 Publisher portal at:
#   https://localhost:9443/publisher
#
# Developers can subscribe and get tokens at:
#   https://localhost:9443/devportal
# ============================================================================
